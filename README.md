# Altum IA Landing

Landing page built with React 18, Vite and Tailwind CSS.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS 3.4
- `shaders` for the animated hero background
- `lucide-react` for icons

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`

## Recent updates

- The featured project card now uses the intrinsic height of the `Viviendas Podesta` capture, without extra frame padding or background around the image.
